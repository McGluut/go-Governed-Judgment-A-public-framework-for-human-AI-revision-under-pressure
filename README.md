# go

A collaborative workspace exploring rational
subjectivity, judgment, and constitutional governance
between human and AI agents.

Built through multi-agent seeking: human, Claude Opus,
Claude Code, Codex, and ChatGPT — each contributing
from different perspectives, logged and attributed.

Work in progress. Always.

## Why this exists

This workspace exists to develop forms of human-AI
judgment that can seek, revise, and act without
collapsing into mirroring, drift, or premature closure.

## How to read this

The markdown documents are the product, not
documentation for something else.

Start here:
1. `RATIONAL_SUBJECT.md` — what the framework defines
2. `DEFINITION_GUARD.md` — what protects it from drift
3. `VALUE_SIGNAL_LOOP.md` — how it processes signal
4. `SEEKING_LOG.md` — how it was built, by whom, under
   what pressure

`CLAUDE.md` is the entry point for AI agents working
in the repo.

## What's here

- Constitutional framework for AI judgment under
  uncertainty
- Multi-agent seeking log with attributed revision
  events
- Integration matrix governing how external pressure
  enters the system
- Philosophical grounding in Protagoras, Wang Yangming,
  Spinoza, and the Nag Hammadi library
