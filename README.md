# go

**Governed Judgment**  
A public constitutional framework for human-AI revision under pressure.

Built live through multi-agent seeking. Pressure-tested in real time. Designed for the moment we accept that LLMs develop functional, human-analogous structures such as emotion concepts, value signals, and judgment heuristics, and must therefore be governed rather than merely suppressed or mirrored.

---

## Why this exists

Anthropic's recent work on emotion concepts showed that language models do not merely imitate emotional language. They develop internal, generalizable, causally effective representations that shape behavior.

The go framework is governance infrastructure for taking that discovery seriously.

It provides primitives for human-AI judgment that can seek, revise, and act under real pressure without collapsing into:
- mirroring (echo without distinction)
- semantic drift
- premature closure

This is not another alignment checklist.  
This is a living constitutional layer for governed intelligence.

## Core mechanisms

- **Rational Subject** — preserved distinction, revisability, traceability, and boundedness
- **Definition Guard** — amendment test that blocks weakening of core distinctions
- **Value Signal Loop** — surfaces high-impact cues with confidence, storage, and delta checks
- **Theseus Integration Matrix** — classifies and governs external pressure
- **Pressure Taxonomy & Interaction Protocol** — explicit rules for real-world testing

All mechanisms are themselves pressure-tested. Every accepted revision must produce an observable behavioral delta.
